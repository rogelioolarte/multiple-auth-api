# Javadoc Documentation

This branch contains only the documentation generated with **Javadoc** for this project.

## Accessing the Documentation

You can open the `index.html` file in the main folder to view the documentation in your browser.

## Generating Javadoc

If you need to regenerate the documentation, run the following command in the main branch of the project:

```sh
mvn javadoc:javadoc
```

Then, update this branch with the generated content.
